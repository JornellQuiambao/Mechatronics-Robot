Software code for a robot created in a mechatronics lab (ECE 118L) in UC Santa Cruz. The software aspect of the robot was coded in C using a Reactive Programming framework with Events and Services. The hardware of the robot, including sensors and materials are documented in the report. The main goal of the robot was to be able to track a beacon signal, and launch and reload projectiles. Main code is in SkiFALL.X.
